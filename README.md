# simple-clock-NTP
very simple clock:  
- based on [ESP32 C3 Super Mini](https://www.espboards.dev/esp32/esp32-c3-super-mini/)
- with NTP
- 7-seg Clock display
- physical switch for daylight savings time
- auto brightness adjustment at fixed time

## libs:  
NTP: https://github.com/arduino-libraries/NTPClient/tree/master  
7-seg display (TM1637): https://github.com/avishorp/TM1637/tree/master
